# TutorFinder Application
TutorFinder-v2 is an Android mobile application **prototype** with user registration that provides students in need of tutoring sessions and tutors wanting to make their services available to students to find and connect (i.e. schedule tutoring sessions, chat), all in an environment controlled and managed by an administrator role.

## Architecture
The architectural pattern MVVM (Model-View-ViewModel) has been used for building the TutorFinder-v2 Android app.

## Technologies
- Java
- Cloud Firestore
- Realtime database
- Glide
